## Welcome to CONIITI 2026

This quickstart guide will get you from zero to running the CONIITI 2026 conference management platform on your local machine.

<Note>
  **Prerequisites**: Make sure you have Node.js v16 or higher installed on your system.
</Note>

## Quick Start

<Steps>
  <Step title="Clone the Repository">
    Clone the CONIITI 2026 repository to your local machine:
    
    ```bash
    git clone <repository-url>
    cd coniiti-2026
    ```
  </Step>

  <Step title="Install Dependencies">
    Install all dependencies for both frontend and backend:
    
    ```bash
    npm install
    cd backend && npm install
    cd ../frontend/vite-project && npm install
    cd ../..
    ```
  </Step>

  <Step title="Configure Environment Variables">
    Create environment files for both frontend and backend. See [Configuration](/configuration) for detailed setup.

    <Tabs>
      <Tab title="Backend">
        Create `backend/.env`:
        ```bash
        SUPABASE_URL=your_supabase_project_url
        SUPABASE_ANON_KEY=your_supabase_anon_key
        SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
        PORT=3000
        ```
      </Tab>
      <Tab title="Frontend">
        Create `frontend/vite-project/.env`:
        ```bash
        VITE_API_URL=http://localhost:3000/api
        VITE_SUPABASE_URL=your_supabase_project_url
        VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Start Development Servers">
    Run both frontend and backend concurrently from the root directory:
    
    ```bash
    npm run dev
    ```
    
    Or run them separately:
    
    <CodeGroup>
      ```bash Backend
      cd backend
      npm run dev
      ```
      
      ```bash Frontend
      cd frontend/vite-project
      npm run dev
      ```
    </CodeGroup>
  </Step>

  <Step title="Access the Application">
    Open your browser and navigate to:
    
    - **Frontend**: [http://localhost:5173](http://localhost:5173)
    - **Backend API**: [http://localhost:3000/api](http://localhost:3000/api)
    
    <Check>
      You should now see the CONIITI 2026 homepage!
    </Check>
  </Step>
</Steps>

## Key Features

- **Conference Management**: Manage speakers, sessions, and schedule
- **User Registration**: Attendee registration and authentication via Supabase
- **Real-time Updates**: Live synchronization of conference data
- **QR Code System**: QR scanning for event check-ins
- **Dynamic Theming**: Country-specific visual themes (Colombia, México, Italia)
- **Admin Dashboard**: Complete control panel for organizers

## Need Help?

If you encounter any issues:

1. Check the [Installation](/installation) guide for detailed setup
2. Review [Configuration](/configuration) for environment variables
