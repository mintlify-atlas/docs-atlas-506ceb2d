# CONIITI 2026 Platform

CONIITI 2026 is a comprehensive conference management platform designed for the **International Congress of Innovation and Trends in Engineering**. Built with React and Express, it provides a complete solution for managing conferences, speakers, attendees, and event logistics.

## Built With Modern Technology

CONIITI 2026 leverages cutting-edge technologies to deliver a robust and scalable platform:

- **Frontend**: React 19 with TypeScript, Vite, and Tailwind CSS
- **Backend**: Express 5 with TypeScript and modular architecture
- **Database**: Supabase for authentication and data persistence
- **Deployment**: Render for both frontend and backend services
- **Performance**: Built-in monitoring and resource tracking
