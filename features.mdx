### Role-Based Access Control

CONIITI 2026 implements a sophisticated permission system with four distinct roles:

<Steps>
  <Step title="SUPER_ADMIN">
    Full system access including configuration, user management, performance monitoring, and trash recovery
  </Step>
  <Step title="ADMIN">
    Manage conferences, speakers, and view analytics
  </Step>
  <Step title="CONTENT_MANAGER">
    Create and edit conferences and speakers
  </Step>
  <Step title="VIEWER">
    Read-only access to analytics and event data
  </Step>
</Steps>

The role check is enforced at the frontend level:

```typescript
useEffect(() => {
  const session = localStorage.getItem("user_session")
  if (session) {
    const user = JSON.parse(session)
    if (user.role === "SUPER_ADMIN" || user.role === "ADMIN" || 
        user.role === "VIEWER" || user.role === "CONTENT_MANAGER") {
      setUserRole(user.role)
    } else {
      navigate("/") // No permissions
    }
  } else {
    navigate("/login") // No session
  }
}, [navigate])
```

### Trash and Recovery System

Super admins have access to a **papelera (trash bin)** feature that stores deleted conferences for recovery:

```typescript
const handleDeleteConference = async (id: string) => {
  if (!confirm("¿Estás seguro de eliminar esta conferencia?")) return;

  try {
    const { deletePonencia } = await import("../services/api");
    await deletePonencia(id);
    setConferences(conferences.filter((c: any) => c.id !== id));
  } catch (err) {
    console.error("Error deleting conference:", err);
  }
}

const handleRestoreConference = (id: string) => {
  const confToRestore = deletedConferences.find((c: any) => c.id === id)
  if (confToRestore) {
    setConferences([...conferences, confToRestore])
    setDeletedConferences(deletedConferences.filter((c: any) => c.id !== id))
  }
}
```

### Speaker-Conference Synchronization

When creating a conference, speakers are automatically extracted and unified from multiple sources:

```typescript
// Unify speakers from API, conferences, and local storage
const finalSpeakers = [...ponentesData];
const speakerNames = new Set(finalSpeakers.map(s => (s.nombre || s.name)));

// Extract from current conferences
finalConfs.forEach((c: any) => {
  const cSpeakerName = c.speaker?.name || c.speaker?.nombre;
  if (cSpeakerName && !speakerNames.has(cSpeakerName)) {
    finalSpeakers.push({
      id: `sp-auto-${c.id}`,
      nombre: cSpeakerName,
      organizacion: c.speaker.organization || "Independiente",
      bio: c.speaker.bio || "Invitado experto",
      avatar_url: c.speaker.avatar || "/default-avatar.png"
    });
    speakerNames.add(cSpeakerName);
  }
});
```

<Tip>

</Tip>

### Download Functionality

The platform includes a download feature that converts the SVG QR code to PNG:

```typescript
const downloadQR = () => {
  const svg = document.getElementById(`qr-code-svg-${qrModalConf?.id}`);
  if (!svg) return;
  const svgData = new XMLSerializer().serializeToString(svg);
  const canvas = document.createElement("canvas");
  const ctx = canvas.getContext("2d");
  const img = new Image();
  img.onload = () => {
    canvas.width = img.width;
    canvas.height = img.height;
    if (ctx) {
      ctx.fillStyle = "white";
      ctx.fillRect(0, 0, canvas.width, canvas.height);
      ctx.drawImage(img, 0, 0);
      const a = document.createElement("a");
      a.download = `Asistencia-${qrModalConf?.title.replace(/[^a-z0-9]/gi, '_').toLowerCase()}.png`;
      a.href = canvas.toDataURL("image/png");
      a.click();
    }
  };
  img.src = "data:image/svg+xml;base64," + btoa(unescape(encodeURIComponent(svgData)));
}
```

### Real-Time Tracking

The platform uses `PerformanceObserver` API to track resource loading:

```typescript
const observer = new PerformanceObserver((list) => {
  list.getEntries().forEach((entry: any) => {
    if (entry.entryType === 'resource') {
      const isImg = ['img', 'image'].includes(entry.initiatorType) ||
        /\.(jpg|jpeg|png|webp|gif|svg)/.test(entry.name);

      if (isImg) {
        trackEvent('IMAGE_LOAD', location.pathname, entry.name, 
                   Math.round(entry.duration), entry.transferSize);
      }
      trackEvent('RESOURCE_LOAD', location.pathname, entry.name, 
                 Math.round(entry.duration), entry.transferSize);
    }
  });
});

observer.observe({ type: 'resource', buffered: true });
```

The platform supports **Spanish** and **English** through a centralized translation system:

```typescript

import type { Language } from "../utils/i18n"

const [lang, setLang] = useState<Language>(
  (localStorage.getItem("app_lang") as Language) || 'es'
)

const t = (key: keyof typeof translations.es) => getTranslation(key, lang)

// Usage in components
<h1>{t('admin_title')}</h1>
```

<Info>

</Info>

## Real-Time Synchronization

All changes made in the admin panel are immediately visible to attendees through:

1. **LocalStorage persistence** for immediate updates
2. **Custom events** to trigger re-renders across components
3. **API synchronization** for backend persistence

```typescript
useEffect(() => {
  localStorage.setItem("site_conferences", JSON.stringify(conferences))
```
