## Overview

This endpoint creates a new conference in the CONIITI 2026 system. Authentication is required to use this endpoint.

<Warning>
  This endpoint requires authentication. Include a valid authentication token in your request headers.
</Warning>

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

### Body Parameters

<ParamField body="titulo" type="string" required>
  Title of the conference
</ParamField>

<ParamField body="descripcion" type="string" required>
  Detailed description of the conference content and topics to be covered
</ParamField>

<ParamField body="fecha_inicio" type="datetime" required>
  Start date and time of the conference in ISO 8601 format
</ParamField>

<ParamField body="fecha_fin" type="datetime" required>
  End date and time of the conference in ISO 8601 format
</ParamField>

<ParamField body="sala_id" type="string" required>
  ID of the room (sala) where the conference will take place
</ParamField>

<ParamField body="dia_evento_id" type="string" required>
  ID of the event day to which this conference belongs
</ParamField>

## Response

<ResponseField name="data" type="object">
  The created conference object
  
  <Expandable title="Conference Object">
    <ResponseField name="id" type="string">
      Unique identifier for the newly created conference
    </ResponseField>
    
    <ResponseField name="titulo" type="string">
      Title of the conference
    </ResponseField>
    
    <ResponseField name="descripcion" type="string">
      Description of the conference
    </ResponseField>
    
    <ResponseField name="fecha_inicio" type="datetime">
      Start date and time
    </ResponseField>
    
    <ResponseField name="fecha_fin" type="datetime">
      End date and time
    </ResponseField>
    
    <ResponseField name="sala_id" type="string">
      Room identifier
    </ResponseField>
    
    <ResponseField name="dia_evento_id" type="string">
      Event day identifier
    </ResponseField>
    
    <ResponseField name="created_at" type="datetime">
      Timestamp when the conference was created
    </ResponseField>
  </Expandable>
</ResponseField>

## Response Codes

<ResponseField name="201" type="Created">
  Conference created successfully
</ResponseField>

<ResponseField name="401" type="Unauthorized">
  Authentication required - Missing or invalid authentication token
</ResponseField>

<ResponseField name="403" type="Forbidden">
  Permission denied - Database constraint error or insufficient permissions
  
  <Expandable title="Error Response">
    <ResponseField name="error" type="string">
      Error message describing what went wrong (e.g., invalid foreign key, duplicate entry)
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST https://api.coniiti2026.com/api/ponencias \
  -H "Authorization: Bearer YOUR_AUTH_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "titulo": "Blockchain y su Impacto en la Industria",
    "descripcion": "Una exploración profunda de cómo blockchain está transformando diferentes sectores industriales",
    "fecha_inicio": "2026-04-16T14:00:00Z",
    "fecha_fin": "2026-04-16T15:30:00Z",
    "sala_id": "sala-002",
    "dia_evento_id": "dia-002"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.coniiti2026.com/api/ponencias', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_AUTH_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    titulo: 'Blockchain y su Impacto en la Industria',
    descripcion: 'Una exploración profunda de cómo blockchain está transformando diferentes sectores industriales',
    fecha_inicio: '2026-04-16T14:00:00Z',
    fecha_fin: '2026-04-16T15:30:00Z',
    sala_id: 'sala-002',
    dia_evento_id: 'dia-002'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_AUTH_TOKEN',
    'Content-Type': 'application/json'
}

payload = {
    'titulo': 'Blockchain y su Impacto en la Industria',
    'descripcion': 'Una exploración profunda de cómo blockchain está transformando diferentes sectores industriales',
    'fecha_inicio': '2026-04-16T14:00:00Z',
    'fecha_fin': '2026-04-16T15:30:00Z',
    'sala_id': 'sala-002',
    'dia_evento_id': 'dia-002'
}

response = requests.post(
    'https://api.coniiti2026.com/api/ponencias',
    headers=headers,
    json=payload
)

data = response.json()
print(data)
```
