---
title: 'Update Conference'
api: 'PUT /api/ponencias/{id}'
description: 'Update an existing conference (ponencia) in the CONIITI 2026 system'
---

## Overview

This endpoint updates an existing conference identified by its ID. Authentication is required to use this endpoint.

<Warning>
  This endpoint requires authentication. Include a valid authentication token in your request headers.
</Warning>

## Request

### Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the conference to update
</ParamField>

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

### Body Parameters

<Info>
  All body parameters are optional. Only include the fields you want to update.
</Info>

<ParamField body="titulo" type="string">
  Updated title of the conference
</ParamField>

<ParamField body="descripcion" type="string">
  Updated description of the conference content
</ParamField>

<ParamField body="fecha_inicio" type="datetime">
  Updated start date and time in ISO 8601 format
</ParamField>

<ParamField body="fecha_fin" type="datetime">
  Updated end date and time in ISO 8601 format
</ParamField>

<ParamField body="sala_id" type="string">
  Updated room (sala) ID where the conference will take place
</ParamField>

<ParamField body="dia_evento_id" type="string">
  Updated event day ID
</ParamField>

## Response

<ResponseField name="data" type="object">
  The updated conference object
  
  <Expandable title="Conference Object">
    <ResponseField name="id" type="string">
      Unique identifier of the conference
    </ResponseField>
    
    <ResponseField name="titulo" type="string">
      Title of the conference
    </ResponseField>
    
    <ResponseField name="descripcion" type="string">
      Description of the conference
    </ResponseField>
    
    <ResponseField name="fecha_inicio" type="datetime">
      Start date and time
    </ResponseField>
    
    <ResponseField name="fecha_fin" type="datetime">
      End date and time
    </ResponseField>
    
    <ResponseField name="sala_id" type="string">
      Room identifier
    </ResponseField>
    
    <ResponseField name="dia_evento_id" type="string">
      Event day identifier
    </ResponseField>
    
    <ResponseField name="updated_at" type="datetime">
      Timestamp when the conference was last updated
    </ResponseField>
  </Expandable>
</ResponseField>

## Response Codes

<ResponseField name="200" type="Success">
  Conference updated successfully
</ResponseField>

<ResponseField name="401" type="Unauthorized">
  Authentication required - Missing or invalid authentication token
</ResponseField>

<ResponseField name="403" type="Forbidden">
  Permission denied - Database constraint error or insufficient permissions
  
  <Expandable title="Error Response">
    <ResponseField name="error" type="string">
      Error message describing what went wrong (e.g., invalid foreign key, conference not found)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="404" type="Not Found">
  Conference with the specified ID does not exist
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X PUT https://api.coniiti2026.com/api/ponencias/987e6543-e21b-34d5-c678-426614174111 \
  -H "Authorization: Bearer YOUR_AUTH_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "titulo": "Blockchain y su Impacto en la Industria - Actualizado",
    "fecha_inicio": "2026-04-16T15:00:00Z",
    "fecha_fin": "2026-04-16T16:30:00Z"
  }'
```

```javascript JavaScript
const conferenceId = '987e6543-e21b-34d5-c678-426614174111';

const response = await fetch(`https://api.coniiti2026.com/api/ponencias/${conferenceId}`, {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer YOUR_AUTH_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    titulo: 'Blockchain y su Impacto en la Industria - Actualizado',
    fecha_inicio: '2026-04-16T15:00:00Z',
    fecha_fin: '2026-04-16T16:30:00Z'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

conference_id = '987e6543-e21b-34d5-c678-426614174111'

headers = {
    'Authorization': 'Bearer YOUR_AUTH_TOKEN',
    'Content-Type': 'application/json'
}

payload = {
    'titulo': 'Blockchain y su Impacto en la Industria - Actualizado',
    'fecha_inicio': '2026-04-16T15:00:00Z',
    'fecha_fin': '2026-04-16T16:30:00Z'
}

response = requests.put(
    f'https://api.coniiti2026.com/api/ponencias/{conference_id}',
    headers=headers,
    json=payload
)

data = response.json()
print(data)
```

## Error Examples

### Conference Not Found

```json
{
  "error": "No rows updated. Conference with id '123' does not exist."
}
```

### Invalid Foreign Key

```json
{
  "error": "update on table 'ponencia' violates foreign key constraint 'ponencia_sala_id_fkey'"
}
```

### Invalid Date Range

```json
{
  "error": "fecha_fin must be after fecha_inicio"
}
```

## Best Practices

<Tip>
  - Only send the fields that need to be updated to minimize data transfer
  - Validate date ranges on the client side before sending the request
  - Ensure the new room (sala_id) has sufficient capacity if changing rooms
  - Check for scheduling conflicts with other conferences in the same room
</Tip>
