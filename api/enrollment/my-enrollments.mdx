---
title: 'Get My Enrollments'
api: 'GET /api/inscripciones/mis-inscripciones'
description: 'Retrieve all enrollments for the authenticated user'
---

This endpoint returns a list of all presentations the authenticated user has enrolled in, including complete presentation details.

## Authentication

This endpoint requires authentication. Include a valid JWT token in the Authorization header.

```
Authorization: Bearer <token>
```

## Response

Returns an array of enrollment objects, each containing the enrollment details and the associated presentation information.

<ResponseField name="id" type="string">
  Unique identifier for the enrollment record
</ResponseField>

<ResponseField name="usuario_id" type="string">
  ID of the enrolled user
</ResponseField>

<ResponseField name="ponencia_id" type="string">
  ID of the presentation
</ResponseField>

<ResponseField name="created_at" type="string">
  Timestamp when the enrollment was created
</ResponseField>

<ResponseField name="ponencia" type="object">
  Complete presentation details
  
  <Expandable title="ponencia properties">
    <ResponseField name="id" type="string">
      Unique identifier for the presentation
    </ResponseField>
    
    <ResponseField name="titulo" type="string">
      Title of the presentation
    </ResponseField>
    
    <ResponseField name="descripcion" type="string">
      Description of the presentation
    </ResponseField>
    
    <ResponseField name="fecha" type="string">
      Date of the presentation
    </ResponseField>
    
    <ResponseField name="hora_inicio" type="string">
      Start time of the presentation
    </ResponseField>
    
    <ResponseField name="hora_fin" type="string">
      End time of the presentation
    </ResponseField>
    
    <ResponseField name="sala_id" type="string">
      ID of the room where the presentation will take place
    </ResponseField>
    
    <ResponseField name="dia_evento_id" type="string">
      ID of the event day
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl -X GET https://api.coniiti.com/api/inscripciones/mis-inscripciones \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## Notes

- The endpoint automatically filters enrollments based on the authenticated user's ID
- Each enrollment includes the full presentation details through the nested `ponencia` object
- Returns an empty array `[]` if the user has no enrollments
- The results are not paginated and return all enrollments for the user
