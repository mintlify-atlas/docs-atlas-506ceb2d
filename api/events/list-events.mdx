---
title: "List Events"
api: "GET /api/eventos"
description: "Retrieve a list of all events for CONIITI 2026"
---

## Endpoint

```
GET /api/eventos
```

## Authentication

This endpoint is **public** and does not require authentication.

## Request

No parameters required.

## Response

<ResponseField name="data" type="array">
  Array of event objects
  
  <ResponseField name="id" type="string">
    Unique identifier for the event
  </ResponseField>
  
  <ResponseField name="nombre" type="string">
    Name of the event
  </ResponseField>
  
  <ResponseField name="descripcion" type="string">
    Description of the event
  </ResponseField>
  
  <ResponseField name="fecha_inicio" type="string">
    Start date and time of the event (ISO 8601 format)
  </ResponseField>
  
  <ResponseField name="fecha_fin" type="string">
    End date and time of the event (ISO 8601 format)
  </ResponseField>
  
  <ResponseField name="ubicacion" type="string">
    Location of the event
  </ResponseField>
  
  <ResponseField name="capacidad" type="integer">
    Maximum capacity for the event
  </ResponseField>
  
  <ResponseField name="created_at" type="string">
    Timestamp when the event was created
  </ResponseField>
  
  <ResponseField name="updated_at" type="string">
    Timestamp when the event was last updated
  </ResponseField>
</ResponseField>
